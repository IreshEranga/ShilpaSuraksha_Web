## 🌟 Overview

**ShilpaSuraksha** is a full-stack, AI-driven web platform developed as a Final Year Research Project at the **Sri Lanka Institute of Information Technology (SLIIT)**. It integrates four intelligent modules under a unified teacher interface to identify and support academically at-risk primary school students in Sri Lanka.

## 🎯 The Problem

Primary school teachers in Sri Lanka manage large classrooms with **limited analytical tools**. By the time a student's academic decline is noticed, critical learning milestones have already been missed. The challenge is compounded when **emotional and behavioral factors** go completely undetected.

### 🔍 Research Gap

<details>
<summary><b>❌ No Unified Multi-Dimensional System</b></summary>
<br/>
Existing systems in Sri Lanka address only academic data (marks, attendance) in isolation. There is no integrated platform combining academic performance, emotional state, behavioral patterns, and intelligent intervention for primary school teachers.
</details>

<details>
<summary><b>❌ Lack of Local Context</b></summary>
<br/>
International EdTech solutions are not adapted for the Sri Lankan primary curriculum, grading structures, or Sinhala/Tamil language contexts. There is an acute absence of locally-trained ML models for Sri Lankan student behavioral data.
</details>

<details>
<summary><b>❌ Reactive Rather Than Predictive</b></summary>
<br/>
Current school management systems (if any) are purely administrative. Teachers identify struggling students only after grades deteriorate — leaving little time for effective intervention before assessments.
</details>

<details>
<summary><b>❌ No Automated Learning Path Generation</b></summary>
<br/>
No system automatically generates personalized remedial learning paths for primary students based on specific subject weaknesses identified via AI, especially within the Sri Lankan curriculum framework.
</details>

---

## 💡 Our Solution

**ShilpaSuraksha** (Shield of Protection) proposes a research question:

> *How can an integrated AI-driven system identify academically at-risk primary school students in Sri Lanka through multi-dimensional analysis — combining academic performance data, emotional state detection from handwriting and drawings, and behavioral patterns — to provide teachers with timely, actionable, and personalized intervention strategies?*

The platform enables:
- 🏫 **School Admins** to manage schools and teachers
- 👩‍🏫 **Teachers** to input academic/behavioral data and receive AI-generated risk flags
- 🎯 **Automated generation** of personalized learning recommendations per student

### 🎯 Component 01 — Early Warning System *(Iresh)*

Accepts teacher-entered **attendance and assessment marks** for each student. A trained classification model processes this academic data to label students as *academically weak* or *on track*, surfacing at-risk students with supporting data visualizations.

- ✅ Academic data entry (marks & attendance)
- ✅ ML risk classification model (weak / not weak)
- ✅ Visual risk flagging for teachers

---

### 🧠 Component 02 — Emotion & Behavioral Analysis *(Shenori)*

Teachers upload student **hand drawings and handwriting samples**. A trained CNN-based deep learning model classifies behavioral and emotional patterns to flag students showing signs of emotional distress or learning difficulties.

- ✅ Hand drawing & handwriting upload interface
- ✅ CNN-based deep learning classification (85%+ accuracy)
- ✅ Emotional distress and behavioral anomaly detection

---

### 📚 Component 03 — Intelligent Recommendation System *(Kavini)*

Displays consolidated lists of academically and behaviorally at-risk students. Teachers specify weak subjects and sections, and the system **auto-generates personalized learning paths** — activities, tasks, and assessments — with progress tracking.

- ✅ At-risk student consolidation view
- ✅ AI-driven learning path generation
- ✅ Activity/task/assessment recommendations
- ✅ Longitudinal progress tracking

---

### 📊 Component 04 — Student Improvement Dashboard *(Sahan)*

A rich analytics dashboard that **clusters students** by academic, behavioral, emotional, and risk indicators using ML clustering (K-Means, DBSCAN). Provides trend charts, benchmarks, an intervention panel, delivery history, and outcome summaries.

- ✅ ML student clustering
- ✅ Interactive trend visualizations
- ✅ Intervention orchestration panel
- ✅ Outcome summaries & benchmarking

## ⚡ Getting Started

### Prerequisites

```bash
node >= 18.x
python >= 3.11
postgresql >= 14
```

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/IreshEranga/Shilpa-Suraksha.git
cd Shilpa-Suraksha
```

### 2️⃣ Backend Setup

```bash
# Navigate to backend
cd backend

# Create virtual environment
python -m venv venv
source venv/bin/activate        # macOS/Linux
venv\Scripts\activate           # Windows

# Install dependencies
pip install -r requirements.txt

# Configure environment variables
cp .env.example .env
# Edit .env with your PostgreSQL credentials and JWT secret

# Run database migrations
alembic upgrade head

# Start the FastAPI server
uvicorn main:app --reload --port 8000
```

The API will be available at `http://localhost:8000`  
Interactive API docs: `http://localhost:8000/docs`

### 3️⃣ Frontend Setup

```bash
# Navigate to frontend
cd frontend

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be available at `http://localhost:5173`

### 4️⃣ Environment Variables

Create a `.env` file in `/backend`:

```env
DATABASE_URL=postgresql://username:password@localhost:5432/shilpasuraksha
SECRET_KEY=your_jwt_secret_key_here
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=60
```

## 📚 Literature Background

<details>
<summary><b>📖 Early Warning Systems in Education</b></summary>
<br/>

Studies show that early identification of at-risk students using attendance, academic performance, and behavioral data can **reduce dropout rates by up to 30%**. Systems that flag students early allow timely teacher intervention before academic gaps widen significantly.
</details>

<details>
<summary><b>🧠 Behavioral & Emotional Analytics</b></summary>
<br/>

Research in child psychology and educational neuroscience confirms that handwriting analysis, drawing patterns, and behavioral cues are reliable indicators of a child's emotional and cognitive state. CNN-based models have demonstrated **85%+ accuracy** in classifying emotional states from drawings.
</details>

<details>
<summary><b>📚 Personalized Learning Paths</b></summary>
<br/>

Adaptive learning systems that tailor educational content to individual student weaknesses show significantly better outcomes than generalized curricula. NLP-driven recommendation engines and knowledge graphs have been used successfully in EdTech platforms globally.
</details>

<details>
<summary><b>📊 Student Clustering & Dashboards</b></summary>
<br/>

Machine learning clustering techniques (K-Means, DBSCAN) applied to academic datasets enable teachers to visualize performance distributions and intervene at the group level. Interactive dashboards combining academic and behavioral metrics **improve teacher decision-making efficiency**.
</details>
